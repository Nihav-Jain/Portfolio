A 2-week rapid prototype in Unity on the theme of storytelling.

  • Developed an API in C# (for Unity) to control the flow of branched dialogue trees and scripted cinematic events. The tree follows the pattern of a behavior-tree, with nodes like sequence, selector, condition etc to determine the flow through the scene.
  • Developed a visual front-end tool, in HTML5 SVG and JavaScript, to help create the dialogue trees in the format required by the API. Check it out here
