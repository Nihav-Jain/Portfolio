The game was presented at the Intel University Games Showcase 2017

The Channeler is a capstone project at FIEA undertaken by 13 graduate students.
Utilizing eye-tracking technology, the development team hopes to innovate with new and exciting mechanics that take advantage of human reaction speed and eye reflexes. We are currently using the Tobii EyeX to track the players eye gaze. My contribution to the project so far -

  • Extended the Tobii EyeX Unreal C++ plugin (for eye tracking) to detect blinking, status of individual eyes, head tracking and modified the plugin interface to be more friendly for gameplay programmers to use
  • Developed data visualizers in C# for all the collected gameplay analytics like heat maps and other puzzle specific analytic questions
  • Wrote a "storytelling" API in C++ for Unreal Engine to control the flow of scripted events and branched dialogue trees
  • Also created a visual front-end tool for the same, using HTML5 SVG, wherein the producers could draw the scene with branching conditions and references to dialogue audios. Check it out here.
  • As the Lead Programmer of the project, worked with the other team leads on planning sprints, game design decisions and managing tasks for the other programmers
  • Also conducted weekly code reviews with the programmers and technical designers
