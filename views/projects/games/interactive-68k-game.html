Interactive Game Project in 68k Assembly

Check out the code for the game

An interactive game developed in Motorola M68000 Assembly Language as part of a 2 week project at FIEA.
Challenges -

  • The game required the physics and graphics engine to be coded in assembly from scratch
  • Since emulators are slower than the actual hardware, the game had to optimized enough to run smoothly on a M68000 emulator like Easy68k
  • Coded a dynamic physics environment from scratch in 68k
  • Coded a 7-segment LED display system to present the score
